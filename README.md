# RF-Dashboards
Contains Grafana and other source files for RF Dashboards

This repo is used for New England AREDN RF Dashboard Development. Dashboards created here are hosted in production at AB1OC. Prometheus currently provides backend time series database support. Please contact AB1OC at [ab1oc@arrl.org](mailto:ab1oc@arrl.org) if you'd like to have your RF links added to this Dashboard.

You can view the current production RF Dashboard at [https://mvara-dash.n1mva.org](https://mvara-dash.n1mva.org).

## Problems and Enhancements

You can submit problem reports and suggest feature enhancements via the standard GitHub Issues mechanisms. Templates are provided. Please limit the Issues you post to topics directly related to the RF Dashboards in this repo.

## Submitting Changes

You can contribute fixes and changes to the files in this repo via GitHub Pull Requests. Please document the nature and scope of your proposed changes. It is best to submit individual changes rather than bundles, as this makes it more likely that we can accept a portion of your changes without requiring a subset of the bundle to be resubmitted.
